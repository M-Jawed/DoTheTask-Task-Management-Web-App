DoTheTask is a simple CRUD todo web app where users can come and create boards, tasks and columns. This project was inspired by the project from the frontend mentor ( Kanban task management web app ), I added my own stylings and tweaks.

### Watch the demo video here: https://www.youtube.com/watch?v=GqaKgc5mBI0

## Features
- Users can create boards
- Users can create columns
- Users can create tasks
- Users can delete and edit tasks, columns and boards
- Darkmode
- Toggle sidenav

## Tech Stack
- Framework: Nextjs
- Backend: Expressjs
- Database: Supabase
- Styling: TailwindCSS
- Icons: React Icons

## Getting started
- Clone the respository
   - git clone https://github.com/M-Jawed/DoTheTask-Task-Management-Web-App
   - cd dothetask
- Install the dependencies
  - npm install
- Go to the backend folder
  - cd dothetask then cd backend
- Install the dependencies in the backend folder
  - npm install
- Start the backend server
   - npm run dev
- Start the server
   - npm run dev (inside dothetask)
